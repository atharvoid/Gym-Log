## Data Safety Declarations

- **Email address** — collected for authentication and cloud sync.
- **Workout/fitness data** — stored locally by default; synced to the cloud when the user is signed in.
- **Device identifiers** — collected by RevenueCat, Google Play, and Sentry crash reporting.
- **Crash logs** — collected by Sentry for diagnosing stability issues.

## Notes

- In-app account deletion is enabled inside the app (Settings → Delete account).
- The web fallback URL is provided for users who have uninstalled the app.
- No lifetime plan is mentioned in store copy.
