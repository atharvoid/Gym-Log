## Project Quick Facts
- **Framework**: Flutter
- **State Management**: Riverpod (`flutter_riverpod`, `riverpod_annotation`)
- **Local Storage**: Drift (SQLite)
- **Backend Auth**: Supabase
- **Routing**: GoRouter
- **Design Target**: OLED-First, High-Density Tracker
